# AI-Powered Multi-Input Article Summarizer using Transformer-Based NLP

## Overview
This project presents an AI-powered article summarization system developed using Natural Language Processing (NLP) and transformer-based deep learning models. The application is capable of generating concise and meaningful summaries from multiple input sources including plain text, PDF documents, and website articles.

The system utilizes the pretrained BART transformer model from Hugging Face Transformers to perform abstractive text summarization efficiently and accurately.

---

## Key Features
- Summarization of lengthy text articles
- PDF document summarization
- Website/article URL summarization
- Transformer-based abstractive summarization
- AI-generated concise summaries
- Error handling and optimized text processing
- User-friendly workflow

---

## Technologies Used
- Python
- Hugging Face Transformers
- BART Transformer Model
- PyPDF2
- Newspaper3k
- Google Colab

---

## Project Workflow
1. User provides input as plain text, PDF, or website URL.
2. Textual content is extracted and preprocessed.
3. The pretrained BART transformer model processes the extracted content.
4. Important contextual information is identified automatically.
5. A concise and meaningful summary is generated and displayed.

## Conclusion

The project successfully demonstrates the implementation of an AI-powered multi-input article summarization system using Natural Language Processing (NLP) and transformer-based deep learning models. The developed system is capable of generating concise, meaningful, and readable summaries from multiple input sources including plain text, PDF documents, and website articles.

The project utilizes the pretrained BART transformer model from Hugging Face Transformers to perform abstractive text summarization efficiently without requiring manual model training. Additional functionalities such as PDF text extraction, website article extraction, formatted summary display, and error handling were integrated to improve the overall usability, stability, and performance of the system.

Through this project, important NLP concepts such as transformer architectures, abstractive summarization, text preprocessing, and real-world text analysis workflows were explored and implemented practically. The developed system significantly reduces the manual effort required to read and understand lengthy textual content while improving information accessibility and efficiency.

The project also demonstrates the practical application of Artificial Intelligence in automated document analysis and content summarization systems. Overall, the developed tool provides a simple, scalable, and effective solution for summarizing large amounts of textual information across different input formats.
